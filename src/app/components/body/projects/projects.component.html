Stock Market Trading Bot

What it is:

This is a Python bot that is running on a personal home server. The bot runs 2 scripts per day- one to gather data and train a machine learning model, the other to execute a buy/sell trade

What it is built with:

  • Mongo DB
  • SMTP Email Protocol
  • Robin-Stocks libray to interact with Robinhood API
  • Tensorflow
  • FinnHub Finance Data API

What I learned:

  • Setting up home servers (port forwarding, firewalls, IP, network programming)
  • Building Machine Learning models with Tensorflow and Keras
  • Mongo DB as a No-SQL way to store data and interact with
  • SMTP email protocol (built into Python) to handle updates from the bot
  • Python requests library to use APIs

Future plans:

  • Deploy on server
  • Make FaceBook Bot to chat and send updates
  • Django Dashboard
  • Refine Tensorflow models and tune hyperparameters
  • Create Dockerfile and run in docker container
